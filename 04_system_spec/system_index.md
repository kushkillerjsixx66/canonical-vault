# System Specification Index  
*Unified architectural specification of the Lattice*

The System Specification defines the **complete architecture** of the Lattice, including:

- kernel modules  
- system modules  
- interfaces  
- constraints  
- lifecycle  
- invariants  
- envelopes  

This index provides a unified entry point into the system specification.
