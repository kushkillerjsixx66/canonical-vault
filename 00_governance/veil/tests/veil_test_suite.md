# Veil Test Suite  
*Canonical enforcement suite for the Veil subsystem*

This suite validates:

- veil levels  
- veil profiles  
- veil enforcement  
- redaction + compression  
- runtime hooks  
- Vault rules  
- silence protocol  
- surface matrix compliance  
- drift detection  
- lineage emission  

The suite is divided into:

1. Unit Tests  
2. Integration Tests  
3. Boundary Tests  
4. Failure Mode Tests  
5. Drift Tests  
6. Silence Protocol Tests  
7. Vault Tests  
8. Surface Matrix Tests  

All tests must pass before the Veil subsystem is considered stable.
