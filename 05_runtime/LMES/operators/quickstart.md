# LMES Quickstart (60 Seconds)

---

## Invoke

```
<Lattice:Run>
Goal: [one clear, bounded sentence]
Constraints: [what's in scope / what's out of scope]
</Lattice:Run>
```

---

## Follow the Protocol

| Step | What Happens | Your Action |
|------|-------------|-------------|
| FRAME | System restates your goal | Read it. Check for misreads. |
| CONSTRAIN | System declares scope | Read it. Check for gaps. |
| **CP1** | System pauses | **Confirm or Adjust** |
| REASONING | System produces numbered steps + sources | Read them. |
| **CP2** | System pauses | **Deepen / Redirect / Stop** |
| FINALIZATION | System produces final output | Read it. |
| **CP3** | System pauses | **Accept / Revise / Reject** |
| AUDIT | System emits Audit Log | Review it. |
| EXIT | Runtime closes | Done. |

---

## Three Checkpoints. Three Decisions.

**CP1:** Is the goal and scope right before we reason?

**CP2:** Are the reasoning steps going in the right direction?

**CP3:** Is the output acceptable?

---

## That's It.

The system handles structure. You handle decisions.
