# Operator Mandate Protocol (OMP)
Version: 0.1 (Draft)
Status: Uninstantiated
Authority: Operator Sovereignty Layer

## Purpose
Define the operator’s mandate: obligations, prohibitions, responsibilities, authority limits, and non-delegable duties.

## Components
- Operator obligations
- Operator prohibitions
- Operator responsibilities
- Sovereignty boundaries
- Non-delegable duties
- Authority limits
- Mandate triggers
- Mandate constraints

## Notes
OMP governs why the operator acts. All other operator protocols govern how.
